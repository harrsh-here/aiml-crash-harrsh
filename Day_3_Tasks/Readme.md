# AIML Crash Course - Project Workspace

This repository houses all programming deliverables and script portfolios completed during the course.

## 📅 Folder Timeline

* **`Day_1_Tasks/`** - Code practice sheet covering syntax, calculations, user inputs, and game states.
* **`Day_3_Tasks/`** - Advanced tasks covering Object-Oriented Programming (OOP), file streams, and Pandas.
