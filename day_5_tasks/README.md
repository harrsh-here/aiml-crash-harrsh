# AIML Course - Day 5 Pandas & NumPy Essentials

This folder contains the 9 core practice deliverables focusing on Python intermediate styling, tabular data manipulation with Pandas, and vectorized matrix computing via NumPy. All tasks are completed using Jupyter Notebooks.

## 📁 Jupyter Notebooks Deliverables

* `task1_profile_card.ipynb` - Builds a typed learner profile card using f-strings and function type hints.
* `task2_json_report.ipynb` - Extracts, filters, and transforms a local profile JSON file using list comprehensions.
* `task3_class.ipynb` - Models an intern workflow pipeline tracking real-world item variables and task states.
* `task4_filter_columns.ipynb` - Loads a student CSV dataset to perform row filters and column selections using boolean indexing.
* `task5_loc_iloc.ipynb` - Demonstrates index selection differences between label-based (.loc) and positional (.iloc) lookups.
* `task6_missing_values.ipynb` - Identifies missing dataset values and handles them using dropna() and fillna() parameters.
* `task7_insights.ipynb` - Produces statistical summaries using describe() and categorical value_counts() groupings.
* `task8_numpy_arrays.ipynb` - Generates, inspects, and subsets multi-dimensional arrays using indexing and negative slicing.
* `task9_numpy_advanced.ipynb` - Implements boolean masking arrays, element broadcasting, and an optimized Cosine Similarity metric.
