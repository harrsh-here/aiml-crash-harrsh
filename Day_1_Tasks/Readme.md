# AIML Crash Course - Day 1 Core Practice Tasks

This workspace establishes our initial hands-on Python programming tasks. These scripts focus on interactive command-line operations, dictionary data structures, conditional flow control blocks, error-safe loops, and basic algorithmic games.

---

## 📁 Day 1 Workspace Directory

| File Name | Description | Command to Run |
| :--- | :--- | :--- |
| `intro.py` | Formats personal user profiles inside structural dictionary containers using clean f-strings. | `python Day_1_Tasks/intro.py` |
| `skills_counter.py` | Maps and iterates arrays of skillset listings utilizing numbered list enumeration counters. | `python Day_1_Tasks/skills_counter.py` |
| `even_odd.py` | Handles user input validation pipelines using try/except blocks to parse numeric parity safely. | `python Day_1_Tasks/even_odd.py` |
| `tip_calculator.py` | Formulates percentage-based service fee amounts returned securely through multi-valued dictionaries. | `python Day_1_Tasks/tip_calculator.py` |
| `word_frequency.py` | Tokenizes user text sentences to aggregate and sort unique item string counts by distribution density. | `python Day_1_Tasks/word_frequency.py` |
| `calculator.py` | Executes standard floating-point arithmetic computations dynamically evaluated by terminal operators. | `python Day_1_Tasks/calculator.py` |
| `grade_classifier.py` | Evaluates student scoring dictionaries inside collections using lambda sorting keys. | `python Day_1_Tasks/grade_classifier.py` |
| `guessing_game.py` | Implements a conditional while loop tracking random target integers against a maximum cap of guess attempts. | `python Day_1_Tasks/guessing_game.py` |
