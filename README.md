# AIML Crash Course - Day 1 & 2 Tasks

This repository contains my solutions for the Python practice assignments, organized cleanly under a unified workspace.

## Project Structure
```text
aiml-crash-harrsh/
├── Day_1_Tasks/          # All practice assignment scripts
│   ├── intro.py          # Profile summary using dicts & f-strings
│   ├── skills_counter.py # Numbered list loops using enumerate()
│   ├── even_odd.py       # Input validation with try/except error handling
│   ├── tip_calculator.py # Function structure returning calculated dict data
│   ├── word_frequency.py # Text extraction arrays and frequency maps
│   ├── calculator.py     # Functional dispatch tables mapping operations
│   ├── grade_classifier.py # Sorted custom lambda conditional structures
│   ├── guessing_game.py  # Automated state loops limited to 7 attempts
│   └── contact_book.py   # Case-insensitive nested record queries
├── .gitignore            # Keeps environment files out of GitHub
└── README.md             # Project documentation
```

## How to Run the Scripts
1. Activate the environment:
   ```bash
   .\.venv\Scripts\Activate
   ```
2. Execute any target script:
   ```bash
   python Day_1_Tasks/guessing_game.py
   ```
