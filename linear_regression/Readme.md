# Linear Regression Practical Tasks

This directory contains the practical exercises for building, evaluating, and testing linear regression models on the California housing dataset.

## Directory Structure

- `linear_regression_tasks.ipynb`: Jupyter Notebook containing the full implementation of all 4 tasks:
  - **Task 1**: Baseline linear regression model using all numeric features, train/test split, metrics calculation, and actual vs predicted values scatter plot.
  - **Task 2**: One-feature (Model A) vs Multi-feature (Model B) comparison table.
  - **Task 3**: Stability evaluation across split ratios (80/20, 70/30, 60/40).
  - **Task 4**: Manual vs library metric calculation verification, Median Absolute Error analysis, and sensitivity analysis under artificial prediction errors.
- `housing.csv`: Local copy of the California Housing dataset.
- `Readme.md`: Overview of the task directory.

## How to Run

1. Open Jupyter Lab in the workspace:
   ```bash
   jupyter lab
   ```
2. Navigate to `linear_regression/linear_regression_tasks.ipynb`.
3. Run all cells.
