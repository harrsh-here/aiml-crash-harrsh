# E-Commerce Sales Performance Analysis (Olist)

This directory contains the final deliverables for the Phase 1 Mini Project covering raw data loading, merge validation, data cleaning, and exploratory data analysis (EDA) for the Olist Brazilian E-Commerce dataset.

## Directory Structure

- `EDA_Olist_Final.ipynb`: Jupyter Notebook containing the end-to-end data cleaning, merge checks, and visual exploration of all 12 business questions.
- `data_quality_report.txt`: Automated data profiling and merge integrity check log.
- `data/`: Directory containing aggregated summary tables (CSVs) for each question.

## How to Run the Analysis

1. Ensure Python 3 (Anaconda environment recommended) is active.
2. Open Jupyter Lab/Notebook:
   ```bash
   jupyter lab
   ```
3. Open `EDA_Olist_Final.ipynb` and run all cells to reproduce the findings and view the plots inline.
